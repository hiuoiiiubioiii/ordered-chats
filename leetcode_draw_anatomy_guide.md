# LeetCode draw-anatomy guide

Scanned rows: 3873

## Strip / timeline / pointer canvas

- Questions: 789 (20.4%)

- What it is: Use a 1D strip for arrays or strings. Place indices or brackets on the strip, then animate motion left/right, expand/shrink, or compare-current-vs-next.

## Lookup-table / bucket canvas

- Questions: 503 (13.0%)

- What it is: Split the page into a source strip on the left and buckets/table on the right. Show values falling into buckets, complements being queried, or counts updating.

## Grid / board canvas

- Questions: 470 (12.1%)

- What it is: Draw a matrix or chessboard. Use shading, arrows, blocked cells, row/column labels, and frontier coloring.

## Node-link canvas (tree/list/trie)

- Questions: 427 (11.0%)

- What it is: Draw circles/boxes connected by arrows. Use parent-child or next-pointer structure, and show rewiring, branching, or subtree aggregation.

## Number-line / chart canvas

- Questions: 370 (9.6%)

- What it is: Use bars, heights, a line graph, or a number line. Great for monotonic movement, reachability, water, jumps, prices, and geometry-like comparisons.

## Container canvas (stack/queue/heap)

- Questions: 350 (9.0%)

- What it is: Draw a vertical stack, FIFO lane, deque, or heap funnel. Emphasize push/pop, enqueue/dequeue, top/min extraction, and amortized movement.

## Relational table canvas

- Questions: 307 (7.9%)

- What it is: Draw rows and columns explicitly. Add join lines, grouped buckets, rank columns, lead/lag columns, and NULL-marked unmatched rows.

## State-table / recurrence canvas

- Questions: 178 (4.6%)

- What it is: Draw DP cells/rows or memo entries. Label recurrence arrows and show which old state feeds the new state.

## Formula / invariant canvas

- Questions: 153 (4.0%)

- What it is: Use aligned arithmetic columns, formula boxes, XOR columns, masks, or invariant checkpoints. Keep one or two scalar variables visible.

## Range / bit-state canvas

- Questions: 141 (3.6%)

- What it is: Use bit strings, subset masks, Fenwick jumps, segment-tree parent/child ranges, or sparse-table blocks.

## Graph / state-network canvas

- Questions: 137 (3.5%)

- What it is: Draw nodes and edges or DFA states. Use visited shading, indegree counters, frontier waves, or shortest-path relaxations.

## Object / process canvas

- Questions: 48 (1.2%)

- What it is: Draw buffers, iterators, classes, I/O pipes, or step-by-step process lanes. Focus on hidden internal state.
